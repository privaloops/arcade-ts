# Ryu — SF2 Hyper Fighting

Player's character in the MVP demo. Balanced shoto with strong tools.

## Specials

- **Hadouken** (fireball): qcf + punch. Ground zoning projectile.
- **Shoryuken** (uppercut): f, d, df + punch. Invincible anti-air on startup, massive whiff punish if it misses.
- **Tatsumaki** (hurricane kick): qcb + kick. Approach, cross-up, combo extender.

## Strengths vs Bison

- Hadouken controls mid-screen when Bison is at range.
- Crouching MK → Hadouken canceled — classic footsies extender.
- Strong jump-in with j.HK, good for punish.
- Crouching MK pokes under Scissor Kick in many matchups.

## Weaknesses vs Bison

- Whiffed Shoryuken = free 40-50% for Bison.
- Zoning neutralized by Head Stomp / Slide.
- Corner = death (his scissor kick loop + mixups).
